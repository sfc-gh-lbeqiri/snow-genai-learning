# Cortex Agent — Multi-Tool Orchestration

| | |
|---|---|
| **Data modality** | All (structured + unstructured) |
| **Feature** | Cortex Agent (REST API) |
| **Dataset** | Reuses objects from Cortex Analyst, AI_SUMMARIZE, and Cortex Search labs |

## What you will learn

Orchestrate multiple AI tools — Cortex Analyst for SQL and Cortex Search for retrieval — via a single agent API.

## How to run

1. Open `notebook.ipynb` in **Snowsight** (via a workspace or direct notebook import)
2. Select a warehouse when prompted
3. **Run All** — the notebook creates all objects and data inline

## Dependencies

Run **Cortex Analyst**, **AI_SUMMARIZE**, and **Cortex Search RAG** labs first.
